GAME TITLE: Endless Wave

DESCRIPTION: An irate underwater diver kidnaps a hapless dolphin and uses the aquatic animals fear-driven overspray to repel attacking sea enemies.  

GAME OBJECTIVE: Survive an endless wave of underwater enemies without the undertow sweeping you into a trench.
